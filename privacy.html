This Privacy Policy explains how Samriddhi Associates handles information when a user uses Samriddhi Gmail Sender.

1. Information Used

  • The Gmail address entered by the user
  • Google OAuth tokens issued after consent
  • Recipient names, email addresses, subjects and messages in the local Excel file
  • Files selected as attachments
  • Local sending status such as Pending, Sent or Failed

2. Google User Data and Permission

The application requests https://www.googleapis.com/auth/gmail.send only to send emails prepared and initiated by the user.

It does not request permission to read Gmail inbox messages or contacts, delete or modify existing messages, or access Google Drive or Google Sheets.

3. Purpose of Use

  • Connect the authorised Gmail account
  • Create personalised email content from the local Excel file
  • Attach user-selected files
  • Send email after the user initiates sending
  • Record sending status locally
  • Reconnect the authorised account later

4. Storage and Retention

The Gmail address, OAuth token and Excel data are stored locally on the user's computer. The application does not upload this information to a Samriddhi Associates server.

Local authorization information remains until the user disconnects the account, deletes local application data, or revokes access through Google Account settings.

5. Sharing and Transfer

Samriddhi Associates does not sell Google user data or use it for advertising. Data is not shared with unrelated third parties.

Information is transmitted to Google only as necessary to authenticate the user and send the email requested by the user. Message content and attachments are delivered through Gmail to recipients selected by the user.

6. Google API Limited Use

Use of information received from Google APIs complies with the Google API Services User Data Policy, including its Limited Use requirements.

7. Security

The application uses Google OAuth so the user does not provide a Gmail password to the software. Users should protect their Windows account, local Excel file and computer.

8. User Controls and Deletion

  • Use Disconnect or Change Account in the application
  • Revoke access in Google Account security settings
  • Delete the local Excel file and application data
  • Contact Samriddhi Associates for assistance

9. Children's Privacy

The application is intended for business and professional users and is not directed to children.

10. Policy Changes

This policy may be updated when functionality, law or Google API requirements change. The latest version will be published here.
