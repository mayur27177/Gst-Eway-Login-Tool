Windows Desktop Application

GSTR-2B Reconciliation

Compare authorised Books or Tally purchase records with the GST Portal GSTR-2B Excel file and produce invoice-wise reconciliation reports.

Main functions

  • Import Books or Tally purchase data
  • Import GST Portal GSTR-2B Excel data
  • Compare suppliers by GSTIN
  • Consolidate multiple tax-rate rows invoice-wise
  • Treat credit notes as negative ITC
  • Generate Matched, Only in GSTR-2B and Only in Books reports

Professional reconciliation workflow

Important: The software must be used only with business and GST data that the user is authorised to process.

The application performs reconciliation according to the files and period selected by the user. The final report should be professionally reviewed before return filing, accounting adjustment or ITC action.

How it works

A local workflow designed for monthly or multi-month reconciliation.

Select the Books or Tally data
Choose the authorised purchase-data file or fetch the required period from a locally accessible Tally company where supported.
Select the GSTR-2B Excel file
Choose the file downloaded from the GST Portal for the relevant period.
Run invoice-wise matching
The application consolidates invoice rows and compares GSTIN, document number, date, taxable value and tax amounts according to the configured logic.
Review and export
Review matched and unmatched records, then save the reconciliation workbook in the selected output folder.

Clear reports
