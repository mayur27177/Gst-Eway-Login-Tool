Samriddhi Gmail Sender

Send personalised emails and optional attachments from a Gmail account authorised by the user.

Main features

  • Client Name and Email Address
  • Dear Client Name greeting
  • Custom subject and message
  • Single or multiple attachments
  • Pending, Sent and Failed status

Gmail permission

https://www.googleapis.com/auth/gmail.send

Used only to send messages initiated by the user. The app does not request inbox-reading access.

Buyer-friendly setup

No Google Cloud project or OAuth JSON selection is required for the buyer.

Enter Gmail ID
Connect Gmail
Approve access once
Edit local Excel and send
Important: Google requires the account owner to approve access. Typing an email ID alone cannot authorise an account.
