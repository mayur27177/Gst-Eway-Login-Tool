GST Tool Privacy Policy

Google Sheets data handling for the GST & E-Way Login Tool.

Last updated: 16 July 2026

1. Google permission

The extension may request https://www.googleapis.com/auth/spreadsheets.readonly to view Spreadsheet data authorised by the user.

2. Purpose

Data is used only to search and display authorised GST or E-Way Bill credential records and assist with user-selected login fields.

3. What the extension does not do

  • Does not edit or delete Sheet data
  • Does not sell Google user data
  • Does not use Google data for advertising

4. Storage and sharing

Authorization information may be stored in the browser or extension environment. Google user data is not shared with unrelated third parties.

5. Google API Limited Use

Use of information received from Google APIs complies with the Google API Services User Data Policy, including its Limited Use requirements.

6. User controls

Users may disconnect Google, revoke access, remove the extension and delete local extension data.
