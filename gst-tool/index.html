GST & E-Way Login Tool

Login assistance for authorised users using records imported from a local Excel file.

Main functions

  • Search authorised client records
  • GST Portal login field assistance
  • E-Way Bill login field assistance
  • Licensed access controls

Local Excel privacy

No Google Sheets, Google Drive or Google Sign-In permission is requested.

The user selects an Excel file from the device. Imported GST and E-Way Bill records remain in Chrome extension storage and are not uploaded to the licence server.

How it works

Import Excel
Select the authorised local Excel file.
Search client
Search firm name, GSTIN or user ID.
Select record
Choose the intended account.
Fill fields
Assist with the selected portal login.
