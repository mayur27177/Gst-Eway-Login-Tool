GST & E-Way Login Tool

Login assistance for authorised users maintaining records in their permitted Google Sheet.

Main functions

  • Search authorised client records
  • GST Portal login field assistance
  • E-Way Bill login field assistance
  • Licensed access controls

Google permission

https://www.googleapis.com/auth/spreadsheets.readonly

Used only to view the Google Sheet authorised by the user. It is not used to edit or delete Sheet data.

How it works

Authorise Google
Grant read-only Sheets access.
Search client
Search firm name or user ID.
Select record
Choose the intended account.
Fill fields
Assist with the selected portal login.
