About the Extension

GST & E-Way Login Tool is a Chrome extension developed to help authorised users search and fill their GST Portal and E-Way Bill Portal login credentials quickly.

The extension reads credential records from a Google Sheet authorised by the user and uses those records only for search and login autofill.

Main Features

  • GST Portal login autofill
  • E-Way Bill Portal login autofill
  • Google Sheets read-only integration
  • Firm name and User ID search
  • Licensed user access
  • Google account authorization

Google API Usage

The extension requests read-only access to Google Sheets. It does not create, edit or delete any Google Sheet data.

Google Sheet information is accessed only after the user gives permission. The extension uses the data to display firm records and autofill login fields selected by the user.

Data Security

The extension does not sell user data. Google Sheet data is not used for advertising and is not shared with unrelated third parties.

Users may disconnect their Google account from the extension at any time.
