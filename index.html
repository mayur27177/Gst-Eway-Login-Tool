About the Application

Samriddhi Gmail Sender is a Windows desktop application developed by Samriddhi Associates. It helps an authorised user send personalised emails to recipients listed in a local Excel file.

The user enters a client name, recipient email address, subject, custom message and optional attachments. Email is sent only after the user starts the sending process.

Main Features

  • Personalised greeting using the client name
  • Separate subject and message for every recipient
  • Single or multiple attachments
  • Local Excel-based recipient list
  • Sent and failed status in the local Excel file
  • Google OAuth authorization without storing the Gmail password
  • Disconnect and change Gmail account option

Google API Access

The application requests only https://www.googleapis.com/auth/gmail.send to send email on the user's behalf.

It does not request access to read the inbox, contacts, drafts or email history, and it does not delete or modify existing Gmail messages.

How Authorization Works

  1. The user enters a Gmail address.
  2. The user clicks Connect Gmail.
  3. Google opens its authorization screen in the browser.
  4. The user selects the account and grants permission.
  5. A local token is stored on the user's computer for future connections.
A buyer does not need to create a Google Cloud project or select an OAuth JSON file. Google still requires the Gmail account owner to approve access once.

Data Handling Summary

  • Email content and attachments remain on the user's computer until sent through Gmail.
  • OAuth tokens and the configured Gmail address are stored locally.
  • No Samriddhi Associates server collects Gmail message data.
  • User data is not sold, used for advertising or shared with unrelated third parties.
