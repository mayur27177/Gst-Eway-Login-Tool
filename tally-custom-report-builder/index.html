Tally Custom Report Builder

Create user-defined reports from authorised Tally data by choosing the required fields, filters and reporting period, then export the results to Excel.

Main functions

  • Connect to the authorised company open in locally running Tally
  • Select available data fields and report columns
  • Choose the required reporting period
  • Apply supported ledger, voucher or transaction filters
  • Preview the resulting records before export
  • Export the custom report to a local Excel file

Controlled local analysis

Report verification: The user must verify the active Tally company, selected period, filters, columns and exported totals before relying on the report.

The application reads only the information required for the user-selected report from the locally accessible Tally environment. Generated reports are saved to locations controlled by the user.

How it works

A flexible local workflow for creating the exact report required.

Open the correct Tally company
Keep the authorised company open and make sure local Tally connectivity is enabled.
Choose fields and period
Select the columns, date range and supported data source needed for the report.
Apply filters and preview
Narrow the records with available filters and review the preview before export.
Export to Excel
Save the custom report in a local Excel file and verify its contents.
