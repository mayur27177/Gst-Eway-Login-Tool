GST Return Downloader Privacy Policy

Privacy and data-handling information for the GST Return Downloader desktop application.

Last updated: 16 July 2026

This policy explains how Samriddhi Associates handles information when an authorised user uses GST Return Downloader.

1. Information used by the application

  • GST Portal usernames and passwords entered in or read from the user's local input file
  • Return type, month, financial year and client instructions
  • Downloaded GSTR-1 and GSTR-3B PDF files
  • The download folder selected by the user
  • Software licence information such as licence key, mobile number, Gmail ID, device ID, status, activation date and last validation

2. Purpose of use

GST credentials and return instructions are used only to perform the user-requested login, navigation and return-download workflow for the authorised GST account.

Licence information is used to validate whether the software is active and authorised for the relevant installation or device.

3. Local GST data and downloaded files

The input Excel file and downloaded GST return PDFs are stored on the user's computer or in the folder selected by the user. The application is designed to process these local files for the requested workflow.

Users should protect the Excel file, downloaded returns, Windows account and computer from unauthorised access.

4. Licence validation

Limited licence and device information may be transmitted to the Samriddhi Associates licence-validation service to confirm activation status, authorised device usage and the most recent validation time.

The licence service is not intended to receive the user's GST return PDFs or the contents of the user's GST credential Excel file.

5. Sharing and sale of data

Samriddhi Associates does not sell GST credential data, downloaded return data or licence information for advertising. Information is not shared with unrelated third parties except where necessary to provide licence validation, comply with law or protect the service from misuse.

6. GST Portal interaction

The application interacts with the GST Portal only to perform the login and return-download actions requested by the user. The GST Portal is a third-party government service governed by its own terms, security requirements and availability.

7. Retention and deletion

  • The user controls retention and deletion of the local Excel file and downloaded PDFs.
  • The user may remove the software and local application data from the computer.
  • Licence records may be retained for activation, support, security and compliance purposes.

8. Security

Reasonable steps are taken to limit the software to its stated functions. Users remain responsible for securing GST credentials, the local device and downloaded return documents.

Do not share: GST passwords, licence keys or downloaded return files with unauthorised persons.

9. User responsibility

The user must have lawful authority to access each GST account processed by the software and must ensure that the input data and requested downloads are accurate and permitted.
