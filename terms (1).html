Windows Desktop Application

Bank Statement to Tally

Import bank statement transactions, review and map the required ledgers, and post authorised accounting entries to a locally running Tally company.

Main functions

  • Import bank statement transaction records
  • Review narration, debit, credit and date details
  • Search and select Tally ledgers
  • Map transactions before posting
  • Post selected entries to the active Tally company
  • Track transaction status locally

Controlled posting

Important: The user must review the selected company, bank ledger, transaction mapping and voucher details before posting.

The application does not connect to the user's bank account and does not initiate bank payments. It processes the statement records selected by the user and communicates only with the locally accessible Tally instance for the requested posting function.

How it works

A guided local workflow from statement import to Tally posting.

Open the bank statement data
Select the supported local statement file and confirm the transaction columns.
Connect to Tally
Keep the correct Tally company open and select the bank ledger and transaction period.
Map and review
Search the appropriate ledgers, review each transaction and confirm the intended voucher treatment.
Post selected entries
Send only the reviewed transactions to Tally and verify the resulting vouchers in Tally.
